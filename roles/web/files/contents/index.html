About This Project

Plannies Mate helps make planning data more accessible to everyone in Australia. We maintain scrapers for council websites to collect and standardize planning application data for public use.

This suite of tools helps developers quickly find which scrapers need attention, diagnose issues with existing scrapers, and develop new ones when council websites change.

Note: Most tools require login access. Contact the administrator if you need access.
